# 多线程如何交替打印
如题：
交替打印出字符串1A2B3C4D5E……26Z，这个题改如何实现呢？
### 解法1
我们在学操作系统的时候学过进程的互斥操作，通过P、V原语来进行进程间通信
两个线程启动后，我们首先让t2进入阻塞状态，t1执行打印操作，t1执行完后唤醒t2，再让t2进行打印,如此反复进行即可得到答案
```java
import java.util.concurrent.locks.LockSupport;
public class Thread0608 {

    static Thread t1 = null, t2 = null;

    public static void main(String[] args){
        char[] N = "ABCDEFGHIJKLMNOPQRSTUVWXYZ".toCharArray();
        int[] C = new int[26];

        for(int i = 0;i < 26; i++){
            C[i] = i+1;
        }
        t1 = new Thread(()->{

            for(char c : N){
                System.out.print(c);
                LockSupport.unpark(t2);
                LockSupport.park();
            }
        });
        t2 = new Thread(()->{
            for(int x : C){
                LockSupport.park();
                System.out.print(x);
                LockSupport.unpark(t1);
            }
        });
        t1.start();
        t2.start();
    }
}
```
### 解法2
主要使用信号量机制，使用信号量标记哪个该运行，代码如下所示:
```java
public class Thread0608 {

    enum ReadyToRun {T1, T2}//枚举类型T1代表走线程1，反之线程2

    static volatile ReadyToRun r = ReadyToRun.T1;//使用volatile禁止了指令重排，同时也保证了内存的可见性

    public static void main(String[] args){
        char[] N = "ABCDEFGHIJKLMNOPQRSTUVWXYZ".toCharArray();
        int[] C = new int[26];

        for(int i = 0;i < 26; i++){
            C[i] = i+1;
        }
        new Thread(()->{

            for(char c : N){
                while(r!=ReadyToRun.T1) {}
                System.out.print(c);
                r = ReadyToRun.T2;
            }
        },"t1").start();
        new Thread(()->{
            for(int x : C){
                while(r!=ReadyToRun.T2) {}
                System.out.print(x);
                r = ReadyToRun.T1;
            }
        },"t2").start();
    }
}
```
